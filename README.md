# 基本介绍

mini酷狗即迷你版酷狗音乐播放器，使您的应用或页面也可以使用酷狗来播放指定的歌曲。它集成音乐的搜索、播放等功能，适用于各种和音乐相结合的应用场景。
支持h5、安卓、ios、和微信小程序；

# 插件主要功能：

播放指定的音乐
播放挂件，搜索歌曲页面，歌曲详情页面
丰富的背景音乐播放器api
